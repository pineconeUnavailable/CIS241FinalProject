Projects

I have created many personal projects in my time as a developer. The most noteworthy of these projects is FileMango.
Filemango is "An extensible framework for continuously creating and updating file metadata summaries using configurable external programs" (Spadavecchia, Easton) built for Linux. At a high level it watches the file system and when a file that matches the specified filters is updated, it dispatches an external program that implements a simple protocol that will add the returned tag(s) to the file at the FS level. A toy use case for this would be to automatically keep metadata tags for files in a user's pictures directory up to date with the average color of the picture.
