Interests

Linux

For most of my life I was a Windows user and I accepted its faults. Then I had my system break twice within a 3 month period.
After having that experience, I was ready for a change.


I was originally drawn to Linux because somebody I went to class with at the time used it on an ancient laptop and it managed to run fairly smoothly despite its age.
When I tried it though, what ended up hooking me was the package manager.
I had never considered it a weird process but it feels so natural now to install programs with the command line.
It isn't much of an issue when installing front end programs that have nice websites and good installers but it is a life saver when doing development work.
Case and point: Installing Flutter Toolkit

Software I use*:

  • Package Manager
  • My Desktop Environment
  • My Future Desktop Environment

Filemango

Filemango is a live modular file analysis and tagging program written in go for Linux systems.
Modules can be written in any language and are launched automatically by the main program as specified in the config file.
When a file is changed in a subdirectory of the one the program is configured to watch, it will automatically be queued for analysis with the relevant analysis modules as per the config.
